---
name: feedback_em_dashes
description: Do not overuse em dashes in prose
type: feedback
---

Avoid em dashes. Use them sparingly if at all — the user finds overuse irritating.

**Why:** 23 em dashes in a single page was flagged as obviously excessive.

**How to apply:** Default to periods, commas, or restructured sentences instead. If an em dash genuinely improves clarity, use it, but treat it as a last resort rather than a default connective.
