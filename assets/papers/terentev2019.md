Terentev and Maloshonok (2019) conducted two experiments embedded in pre-course surveys for 28 MOOCs (Massive Open Online Courses), collecting data from approximately 22,910 learners across Russian- and English-language courses. The experiments tested primacy effects in web surveys.

Experiment 1 presented a 5-point vertical item-by-item question about prior familiarity with course topics (a non-attitudinal, factual-type question). A primacy effect was found: respondents were more likely to select options at the top of the vertical list. This effect was moderated by education — less educated respondents showed stronger primacy effects, consistent with satisficing theory.

Experiment 2 presented a 4-point grid (matrix) question about agree/disagree motivation items for taking the course. No primacy effect was found in the grid format, suggesting that the grid presentation reduces susceptibility to positional biases — possibly because the simultaneous visual presentation of all options in the grid allows more complete processing.

**Claim** — A primacy effect was found for a vertical item-by-item question, but not for a grid (matrix) question, suggesting that format moderates response order effects.
**Quote** — "Primacy effects were found in the item-by-item format but not in the grid format."

**Study details**
- Topic/construct: Exp 1: course topic familiarity (non-attitudinal); Exp 2: course motivation (agree/disagree)
- Response options: Exp 1: 5-point vertical item-by-item; Exp 2: 4-point grid agree/disagree
- Scale format: Exp 1: vertical item-by-item; Exp 2: matrix/grid
- Design: Split-ballot between-subjects (order manipulated)
- Sample: N ≈ 22,910 MOOC learners; Russian- and English-language courses
- Mode: Web (MOOC pre-course surveys)
- Main finding: Primacy effect in vertical item-by-item format (especially for low-education respondents); no primacy effect in grid format
