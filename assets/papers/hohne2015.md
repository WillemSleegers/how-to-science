Höhne and Lenzner (2015) used eye-tracking methodology to examine response order effects in web surveys, comparing horizontal and vertical scale layouts. Eighty-four participants at GESIS (Germany) completed a web questionnaire containing 6-point fully labeled agree/disagree scales. Items measuring competition were presented horizontally and items measuring visibility were presented vertically.

Eye-tracking data showed that respondents processed scales from left to right (for horizontal) and top to bottom (for vertical). Response order effects were larger for vertically presented scales than for horizontally presented scales. Eye-tracking evidence showed that respondents tended to skip the last (bottom) categories in vertical scales — consistent with a satisficing explanation. In horizontal scales, the full set of response options received more processing. The study provides process-level evidence (not just aggregate responses) that positional primacy effects are stronger in vertical layouts because respondents are less likely to examine options at the end of the scale.

**Study details**
- Topic/construct: Competition and visibility attitudes
- Response options: 6-point fully labeled agree/disagree scale
- Scale format: Horizontal (competition items) and vertical (visibility items)
- Design: Within-subjects (different items in each format); eye-tracking
- Sample: N = 84 participants at GESIS, Germany
- Mode: Web (laboratory eye-tracking)
- Main finding: Response order effects larger for vertical scales; eye-tracking shows skipping of terminal categories in vertical layouts; supports satisficing
