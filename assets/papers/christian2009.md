Christian, Parsons, and Dillman (2009) report five sets of experiments embedded in two web surveys of undergraduate students at Washington State University (total N across both surveys ≈ 3,156). The surveys evaluated student experiences and were used to test how various aspects of the visual presentation of scalar questions affect responses. Each survey had four experimental versions, and students were randomly assigned to conditions.

**Experiment 1: Category order (positive vs. negative first).** Ten questions tested whether presenting the most positive response option first (vs. most negative first) affected responses. Significant differences in response distributions were found for only 3 of 10 items, and only one difference of means was statistically significant. Critically, significant differences appeared only for items presented vertically (not horizontally). Paradata showed that response times were longer (in 8 of 10 items) when the negative category appeared first, consistent with the "up means good" heuristic — respondents take longer when scale order violates their expectations.

**Experiment 2: Linear vs. nonlinear layout.** Four questions compared a linear layout (all options in one vertical column) to a nonlinear layout (options split across multiple columns). Significant differences were found; respondents were more likely to select the third option in the nonlinear layout, where it appeared first in the second column — consistent with visual position influencing interpretation.

**Experiment 3: Spacing of "Don't Know" and midpoint categories.** Two questions tested whether adding visual space before the "Don't Know" option (separating it from the substantive categories) affected its selection. Adding space before "Don't Know" significantly increased its selection. Isolating the midpoint similarly affected responses.

**Experiment 4: Polar point vs. number box.** Six questions compared a standard polar point scale (graphically displayed with labeled endpoints) to a number box format (respondents type a number). Responses to the number box tended to be more negative. However, when higher numbers were assigned to positive endpoints (consistent with conventional positive valence), differences disappeared or reversed — the number box produced similar or more positive responses.

**Experiment 5: Polar point with vs. without numeric labels.** Four questions compared polar point scales with versus without numeric labels for all categories. Results were generally non-significant, suggesting that adding numeric labels to a fully verbally-labeled polar point scale does not substantially change responses.

**Study details**
- Topic/construct: Student experience ratings (satisfaction, accessibility, desirability, quality)
- Response options: 5-point fully verbalized scales (various: satisfied/dissatisfied, excellent/terrible, etc.); also number boxes and polar points
- Scale format: Web radio buttons, horizontal and vertical; number box; polar point with/without numbers
- Design: Split-ballot between-subjects (4 versions per survey); 5 sets of experiments
- Sample: N ≈ 3,156 WSU undergraduate students across two surveys (2003)
- Mode: Web survey
- Main finding: Category order effects mainly for vertical scales; nonlinear layout creates position bias; spacing affects "don't know" selection; number box produces more negative responses when high numbers assigned to negative pole; numeric labels on polar point have little effect
