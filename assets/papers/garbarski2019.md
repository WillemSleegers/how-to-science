Garbarski, Schaeffer, and Dykema (2019) conducted a 2×3 factorial experiment using Amazon Mechanical Turk (MTurk, N = 2,945) to examine both response option order (excellent-to-poor vs. poor-to-excellent) and scale orientation (vertical, horizontal, or banked/grid) for the standard 5-point SRH question presented in a web survey.

There were significant main effects of response option order: respondents reported better self-rated health when options ran from excellent to poor. There were no significant main effects of scale orientation (vertical vs. horizontal vs. banked). Crucially, there was no statistically significant interaction between response option order and orientation — meaning the response order effect was equally present regardless of how the scale was displayed on screen. The authors conclude that the primacy effect for SRH response options is robust to changes in visual presentation format and cannot be eliminated simply by changing the scale's orientation.

**Claim** — Response option order (excellent-first vs. poor-first) has a significant main effect on SRH, but scale orientation (vertical, horizontal, banked) does not, and the two factors do not interact.
**Quote** — "There were significant main effects of response option order but no main effects of orientation and no interaction."

**Study details**
- Topic/construct: Self-rated health (SRH)
- Response options: 5-point (excellent / very good / good / fair / poor), order manipulated
- Scale format: 3 orientations tested: vertical, horizontal, banked (grid)
- Design: 2×3 factorial between-subjects (response order × orientation)
- Sample: N = 2,945 MTurk workers
- Mode: Web
- Main finding: Primacy effect for response option order robust across all three orientations; orientation itself has no effect
